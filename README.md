🧠 AI Resume Builder

An AI-powered Resume Builder web application that helps users create, edit, enhance, and share professional resumes with ease. Built using the MERN stack with AI integration for smart content enhancement.

🚀 Features 🔐 Authentication

User registration & login (JWT-based)

Secure password hashing using bcrypt Protected routes for authenticated users

📄 Resume Builder

Create and edit resumes dynamically Sections supported:
Personal Information 
Professional Summary
Work Experience
Education
Projects
Skills

Live resume preview while editing

🤖 AI-Powered Enhancements

AI-enhanced Professional Summary

AI-enhanced Job Descriptions

Uses AI prompts to generate ATS-friendly, professional content

🌍 Resume Sharing

Make resume public

Shareable resume link (view-only)

Works without login for public resumes

🎨 Customization

Multiple resume templates Accent color selection Clean, modern UI

📥 Export Resume preview ready for PDF download (extendable)

🛠️ Tech Stack

Frontend

React.js Redux Toolkit Tailwind CSS Axios React Router Lucide Icons

Backend

Node.js Express.js MongoDB + Mongoose JWT Authentication bcrypt

AI

OpenAI API (for text enhancement)
